Aplikasi ini merupakan program sederhana berbasis menu yang memungkinkan pengguna melakukan konversi jarak, berat, dan mata uang (Rupiah, USD, SGD).
Pengguna memilih jenis konversi melalui angka yang mewakili setiap opsi, kemudian menentukan satuan asal, memasukkan nilai yang akan dikonversi, memilih satuan tujuan, dan mendapatkan hasil konversinya.

Fitur Utama

Konversi jarak (misalnya km, m, cm, dll.)
Konversi berat (misalnya kg, g, mg, dll.)
Konversi mata uang:
IDR (Rupiah)
USD (Dollar Amerika)
SGD (Dollar Singapura)

Navigasi berbasis angka (menu pilihan)
Hasil ditampilkan langsung
Pengguna dapat memilih untuk kembali ke menu utama atau keluar
